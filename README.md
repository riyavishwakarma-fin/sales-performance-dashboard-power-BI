# sales-performance-dashboard-power-BI
Power BI dashboard analyzing sales performance, trends, and business insights.📊

📌 Overview

This project presents a Sales Performance Dashboard built using Power BI to analyze business data and generate actionable insights.

🚀 Features

- Sales, Profit & Profit Margin overview
- Monthly sales trend analysis
- Segment-wise performance
- Product-wise insights
- Country-level sales distribution

🛠️ Tools Used

- Power BI
- Data Visualization
- DAX (Basic Measures)

🎯 Objective

To transform raw sales data into meaningful insights using interactive and visually appealing dashboards.
